# live-bus-people-counting

This is python code to detect and count people on tbe bus.

Dependencies:

pip install python-opencv

pip install imutils
    
Usage:

- To use local video file,
        people_counter.py --v[video] test.avi

- To use webcam,
        people_counter.py
